# 「絶対復習」PHPクローン

「絶対復習」というWebアプリのPHPクローンです（オリジナルはRails）。

オリジナルの「絶対復習( http://brushup.narihiro.info/ )」がメンテナンス中になっていることが多くなり、使いづらく感じたので、自分専用の「絶対復習」を作りました。

- テキストを登録できる
- 周期的に復習できる

といった「絶対復習」の特徴は受け継いでいますが、「ユーザー機能」は有していません。

## 技術仕様

- PHP 5.2.8以上（CakePHP2の動作する環境なら動作するはず）
- CakePHP 2.4.2
- MySQL 5.5
